# AI 이미지 프롬프트 시뮬레이터 · 프롬프트 갤러리

같은 장면 하나를 **13가지 스타일**로 생성해, AI 이미지 프롬프트의 "스타일 한 줄" 차이가 결과를 어떻게 바꾸는지 한눈에 비교하는 프롬프트 갤러리입니다.

🔗 **데모:** https://commme.github.io/ai-prompt-simulator/

## 공통 장면
> 창가 근처에서 세라믹 컵을 들고 있는 단발머리 여성, 미디엄샷, 아이레벨, 따뜻한 자연광, 1:1 비율.

이 장면을 고정하고 스타일만 바꿉니다.

## 13가지 스타일
실사형 사진 · 픽사풍 3D · 디즈니풍 동화 · 지브리풍 수채 · 토이 피규어 · 디오라마 · 레고 블록 · 클레이 · 픽셀 아트 · 사이버펑크 · 플랫 일러스트 · 매거진 화보 · 모노톤 필름

## 구성
- `index.html` — 시뮬레이터 갤러리 (이미지가 base64로 내장된 단일 파일, 그대로 열면 동작)
- `스타일_샘플_생성_프롬프트_13종.md` — 직접 생성해 볼 수 있는 13종 프롬프트
- `images/` — 13개 스타일 원본 이미지

## 사용법
`index.html`을 브라우저로 열거나 위 데모 링크에 접속하세요. 별도 설치/빌드가 필요 없습니다.
